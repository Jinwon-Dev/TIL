# TIL
기억보다는 기록이 중요하다고 생각합니다.

좋은 개발자가 되기 위해 그 날의 학습한 내용을 기록합니다.

1. 디렉토리와 파일명은 영문으로 작성합니다.
2. 추가적인 검색의 비용이 들지 않도록 간략하지만 정확히 기록합니다.
3. 당일에 학습한 내용은 당일에 기록합니다.
